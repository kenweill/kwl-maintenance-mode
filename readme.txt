A fully customizable maintenance/under-construction page for WordPress.

== Description ==

KWL Maintenance Mode lets you display a beautiful, branded maintenance page
to visitors while your site is under construction — fully customizable from
the WordPress admin dashboard.

Customize:
* Company / site name
* Tagline and message text
* ETA / status text
* Progress bar (value + colors + toggle)
* Contact links (email, Facebook, check-status button)
* All colors (background, card, title, buttons, progress bar)
* Who can bypass (admins, editors)
* SEO robots meta tag

== Installation ==

1. Upload the plugin ZIP via Plugins > Add New > Upload Plugin
2. Activate the plugin
3. Go to Settings > KWL Maintenance to customize
4. Toggle maintenance mode ON when ready
